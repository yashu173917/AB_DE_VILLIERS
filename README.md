# 🏏 AB de Villiers — Fan Tribute Page

A visually rich, single-page fan tribute to **AB de Villiers**, one of cricket's greatest ever batsmen. Built with HTML, CSS, and Bootstrap 5 — no frameworks or build tools required.

🌐 **Live Site:** [yourusername.github.io/your-repo-name](https://yourusername.github.io/your-repo-name)

---

## ✨ Features

- **Cinematic Hero Section** — Full-screen hero with animated name reveal, tagline, and scroll indicator
- **Career Statistics** — Interactive cards for IPL, ODI, and Test cricket with image backgrounds
- **World Records** — Grid layout showcasing 6 of AB's most iconic records
- **Expanding Gallery** — Hover-to-expand photo slider with smooth CSS transitions
- **Cricket Animation** — Tap the tribute button to trigger a "SIX! 🎉" ball & bat animation
- **Scroll Animations** — Powered by AOS (Animate On Scroll)
- **Dark Theme** — Sleek black and gold design throughout
- **Custom Cursor** — Cricket bat 🏏 emoji cursor
- **Fully Responsive** — Optimised for mobile, tablet, and desktop

## 🚀 Getting Started

### View Locally
Just open `index.html` in any modern browser — no setup or installation needed.

### Deploy to GitHub Pages
1. Upload `index.html` to your GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/repo-name`
